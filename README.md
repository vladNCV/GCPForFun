# GoogleCloudCEDemo

Test project to be build and containerized automatically by google cloud container registry.

Using:
  - Java 9 (openjdk)
  - Project Lombok
  - Junit 5
  - Spring :
    - Boot :
        - Core
        - WebFlux (reactive support)
        - Actuator
        - DevTools
        - Configuration Processor
    - Cloud Config Server
  - Google Cloud :
    - Compute Engine
    - Function
    - Load Balancer & Auto-Scaling
    - Container Builder
